## Features
#### 로그인
사용자 아이디와 비밀번호를 검증하고 접근을 허용합니다.

---
#### 기준 관리
HR 도메인에 대한 기준을 정하고 관리합니다.

- 부서 기준 관리
  - 부서에 대한 기준을 정하고, 이를 관리합니다.

---
#### 인사 관리
인사 시스템에 필요한 부서와 사원을 관리합니다.

- 부서 관리
  - 기준 관리에서 등록한 부서 기준에서 부서를 등록 및 관리합니다.
- 부서원 관리
  - 부서 관리에서 등록한 부서에 사원을 배치하고 관리합니다.
- 사원 관리
  - 사원 추가, 수정, 삭제 등을 통하여 사원 정보를 관리합니다.
